\#\# 3.2 Required Interactions

\#\#\# Animations (Subtle, Premium)  
| Element | Animation | Trigger |  
|---------|-----------|---------|

| Navigation | Glassmorphism background | On scroll (past hero) |  
| Timeline items | Fade-in \+ slide from left/right | On scroll into view |  
| Project cards | Lift \+ shadow increase | On hover |  
| Philosophy cards | Subtle glow | On hover |  
| CTA buttons | Magnetic effect \+ scale | On hover |  
| Stats numbers | Count-up animation | On scroll into view |

\#\#\# Scroll Behavior  
\- Smooth scroll for anchor links  
\- Parallax (subtle) on hero background  
\- Progress indicator (thin line at top)

\#\# 3.3 Responsive Breakpoints

\`\`\`  
Mobile: \< 640px  
Tablet: 640px \- 1024px  
Desktop: 1024px \- 1440px  
Large: \> 1440px  
\`\`\`

\#\#\# Mobile Adaptations  
\- Hero: Stacked layout, smaller type  
\- Timeline: Vertical line, alternating sides  
\- Projects: Single column, stacked cards  
\- Philosophy: 2x2 grid → single column  
\- Stats: 2x2 grid → single column

\#\# 3.4 Performance Requirements

\- First Contentful Paint: \< 1.5s  
\- Time to Interactive: \< 3s  
\- Lighthouse Score: \> 90 (all categories)  
\- Images: WebP format, lazy loading  
\- Fonts: Preload critical fonts, font-display: swap
